# 📊 Exploratory Data Analysis Dashboard

## 🚀 Project Overview

This project focuses on performing **Exploratory Data Analysis (EDA)** using Power BI on an e-commerce dataset. The dashboard was designed to uncover meaningful insights, identify trends and outliers, and support data-driven decision-making through interactive visualizations.

This project demonstrates my skills in:

* Data analysis
* Business intelligence
* Dashboard design
* Data visualization
* Analytical thinking

---

## ✨ Key Features

✔ KPI Cards for:

* 💰 Total Sales
* 📈 Average Sales
* 📊 Median Sales
* 🛒 Total Orders

✔ Interactive Visualizations:

* Sales Trend Analysis
* Product-wise Sales Analysis
* Payment Method Distribution
* Order Status Analysis
* Outlier Detection Table

✔ Interactive Slicers & Filters for dynamic analysis

✔ Clean and user-friendly dashboard design

---

## 🛠 Tools & Technologies Used

* 📊 Microsoft Power BI
* 🔄 Power Query
* 📈 Data Visualization Techniques
* 🧮 DAX Measures
* 📂 Data Cleaning & Transformation

---

## ▶️ How to Run the Project

1. Open the `.pbix` file using Microsoft Power BI Desktop.
2. Refresh the dataset if required.
3. Use the slicers and filters to interact with the dashboard and explore insights.

---

## 🔍 Key Insights

* Certain products generated significantly higher sales compared to others.
* Card and online payment methods contributed major portions of total revenue.
* Delivered orders produced the highest sales performance.
* Some transactions showed unusually high values, indicating potential outliers.
* Sales trends fluctuated over time, revealing valuable business patterns.

---

## 🎯 Project Objective

The objective of this project was to apply EDA techniques to transform raw data into meaningful business insights using interactive Power BI dashboards.
